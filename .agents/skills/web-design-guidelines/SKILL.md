# Web Interface Guidelines Skill

Use this skill when auditing, reviewing, or writing UI code to ensure compliance with modern accessibility, UX, and performance standards.

## Rules
1. **Accessibility**: Use semantic HTML5 elements. Verify `role`, `aria-label`, and interactive keyboard controls. Ensure visible focus rings.
2. **Touch Targets**: Keep interactive areas at a minimum of 44x44px.
3. **Typography**: Ensure appropriate contrast ratios (WCAG AA). Use tabular numbers (`font-mono` / `tabular-nums`) for dynamic metrics.
4. **Layout Thrashing**: Avoid operations that cause unnecessary browser reflows/repaints.
