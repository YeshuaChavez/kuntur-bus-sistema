# Frontend Design Skill

Use this skill when designing or refactoring user interfaces in this project to prevent generic "AI slop" templates.

## Rules
1. **Audience & Context**: Clarify the specific tone and audience (KUNTUR is an premium, modern Andean transportation company) before coding.
2. **Typography**: Set clear type scales, proper weights, and widths. Avoid default system fonts when possible; use project-defined Google Fonts.
3. **Palette**: Stick to defined brand colors and high-contrast, premium dark mode palettes.
4. **Layout**: Break standard grid boundaries when appropriate, using asymmetrical spacing or overlapping elements to add depth.
5. **No Emojis**: Avoid emojis in the UI; prefer structured icon systems (e.g., Lucide Icons).
